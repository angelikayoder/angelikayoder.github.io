What are Ruby classes?

December 28, 2015

Ruby classes are super-easy to understand. Below is a breakdown of some of the main points.

Class Variables

To create Class variables, use the @@ to denote the variable as class level. This allows you to access the variable throughout methods.

class Taxonomic_rank
  @@living_beings
end

Class Level Instance Variables

What is a class? It's an object. What can objects have? Objects can have class and instance variables. This means that a class can have instance variables. Let's add an instance variable:

class Taxonomic_rank
  @hierarchy = 7
end

The attr_accessor method allows you to methods to access methods at the class level rather than the instance level.
