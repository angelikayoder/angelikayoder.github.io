Enumerable Methods- Map

December 11, 2015


Today, I've chosen to write about a very handy Ruby enumerator called "map". I chose this one specifically because although I've used it while doing exercises in the past, I was still unsure of it's full power and all of it's uses. Below are some of the main points I've derived from my research.

A definition for map says, "Returns a new array with the results of running block once for every element in enum."

Below, the code has to call the block given on every item of the collection and then build an array of the results of every call.

def map(&block)
result = []
each do |element|
result << block.call(element)
end
result
end

Map appears to be a method I'll use often in coding.
